# Truck & Trailer Repair App Frontend

This is the Flutter frontend for the Truck & Trailer Repair App.

## Features
- Super Admin, Admin, and Technician panels
- Inventory & vehicle intake
- Job assignment and tracking
- History and logs
- Analytics dashboard
- Modern, unique UI

## Structure
- `lib/screens/` — All main screens (login, signup, dashboards, modules)
- `lib/widgets/` — Reusable widgets
- `lib/models/` — Data models
- `lib/services/` — API and business logic
- `lib/utils/` — Helpers and utilities

## Getting Started
1. Install Flutter SDK
2. Run `flutter pub get`
3. Run the app:
   ```bash
   flutter run
   ```
