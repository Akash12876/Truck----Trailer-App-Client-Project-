# Truck & Trailer Repair App Frontend

This is the Flutter frontend for the Truck & Trailer Repair App.

## Features
- Super Admin, Admin, and Technician login/signup
- Inventory and vehicle intake
- Job assignment and tracking
- Vehicle history and system logs
- Analytics dashboard

## Structure
- `lib/screens/` - Login, inventory, jobs, history/logs, and analytics screens
- `lib/widgets/` - Reusable UI widgets
- `lib/models/` - API data models
- `lib/services/` - API client and backend calls
- `lib/utils/` - Theme and helpers

## Getting Started
1. Install Flutter SDK.
2. Run `flutter pub get`.
3. Run the app:
   ```bash
   flutter run
   ```

The API client defaults to `http://127.0.0.1:8000`.
